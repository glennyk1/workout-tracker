# Workout Tracker
**Workout Tracker** is a Python-based application that makes logging and tracking your workouts simple and efficient.

## Features

- **Log Workout:**
  Log your workout sessions by entering the workout date, exercise, reps, weight, and optionally the RPE.

- **View History:**
  View all logged workouts, with details of each exercise and set.

- **Add Exerise:**
  Add new exercises to your list for tracking.


## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/glennyk1/workout-tracker.git
2. Navigate to the project directory: 'cd workout-tracker'


## Usage

- **CLI Version:** To run the command-line version, execute: 'python workout_tracker.py'
- **GUI Version:** To run the graphical user interface, execute: 'python gui_app.py'


## Technologies Used

Python, Tkinter (for the GUI), Git & Github (for version control)
